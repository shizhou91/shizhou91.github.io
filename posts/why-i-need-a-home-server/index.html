Introduction#

Home server

Writing#

It’s more easy to writing every where. I build a website with Hugo and Github pages. Then I set the set workspace on my linux server. At the same time, I build a visual code server. I could access this code server anywhere. The coder server will save the status of editing. Just continue my article anywhere. Comparing to setup the env in a sepcific laptop, I only could edit and deploy on that laptop.

Save Money on storage for video#

In general, I use Mac mini for personal daily use. The storage price of Mac mini is expensive. Then I could hold my video to my home server. And expose the shared shorage with Samba service. The Apple TV uses the Infuse play the video served by the Samba service. We can watch high qaulity video fluently at home with big TV.

Airplay to normal speaker#

There is a software could enable play music on the ubuntu server. I put my lab on the living room. Then I could use iPhone to play music on it through Airplay 2. In this case, you don’t have to buy a Apple speaker. Comparied with buletooth there are some benefits:

  1. Your speaker don’t have to be support bluetooth or airplay.
